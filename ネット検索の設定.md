# ネット検索の設定

食材を入力すると、**ネット上のレシピ**も一緒に検索できます。

## 標準：TheMealDB（完全無料）

- **登録不要・APIキー不要・クレジットカード不要**
- 約600品のレシピ（主に洋食）を検索可能
- そのまま使えます

## オプション：Edamam（栄養データ付き）

より多くのレシピと**栄養表示**を使う場合は Edamam を設定：

1. [Edamam](https://developer.edamam.com/) でアカウント作成
2. Recipe Search API のキーを取得
3. 環境変数に設定

```
EDAMAM_APP_ID=あなたのApp ID
EDAMAM_APP_KEY=あなたのApp Key
```

Edamam 未設定時は TheMealDB が自動で使われます。
