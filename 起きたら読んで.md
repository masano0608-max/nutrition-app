## やること（30秒）

**「今すぐデプロイ.command」** をダブルクリックしてください。

フォルダ「子供達の栄養管理」の中にあります。

- ダブルクリック → push が実行される
- 2〜3分後に Cloud Build が自動でビルド・デプロイ
- 完了したらアプリが更新されます

---

## うまくいかない場合

**GitHub Desktop** を使っている場合は：

1. GitHub Desktop を開く
2. 「Push origin」をクリック
