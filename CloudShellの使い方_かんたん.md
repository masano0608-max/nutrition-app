# Cloud Shell の使い方（かんたん版）

リポジトリはもう clone できています。あと 2 つのコマンドを打つだけです。

---

## 画面の場所を確認

Cloud Shell の画面の**下の方**に、黒いエリアがあります。ここが「ターミナル」です。

```
┌─────────────────────────────────────────┐
│  上の方：ファイル一覧など                │
├─────────────────────────────────────────┤
│  下の方：ターミナル（黒い画面）          │
│  masano0608@cloudshell:~/...$  ← ここに │
│  コマンドを打ちます                      │
└─────────────────────────────────────────┘
```

---

## ステップ1：黒いエリアをクリック

1. 画面**下の黒い部分**（ターミナル）をクリックする
2. カーソルが点滅していればOK（ここに文字が打てます）

---

## ステップ2：1行目をコピーする

次の文字を**全部**選択してコピーしてください。

```
chmod +x CloudShellでGitHubデプロイ設定.sh
```

💡 コピー方法：上の文字をドラッグして選択 → 右クリック → コピー  
（Macなら ⌘+C でもOK）

---

## ステップ3：貼り付けて Enter

1. 黒いエリア（ターミナル）をクリック
2. **貼り付け**（右クリック → 貼り付け、または ⌘+V）
3. **Enter キー**を押す

何も出なくても大丈夫です。次のステップに進んでください。

---

## ステップ4：2行目をコピーする

次の文字を**全部**選択してコピーしてください。

```
./CloudShellでGitHubデプロイ設定.sh
```

---

## ステップ5：貼り付けて Enter

1. 黒いエリアをクリック
2. **貼り付け**
3. **Enter キー**を押す

---

## 成功すると

「cron-secret を登録...」「IAM 権限を付与...」といった文字が流れて、最後に「✅ セットアップ完了」と出れば成功です。

---

## うまくいかないとき

「No such file or directory」と出る場合：

1. 今いる場所が `nutrition-app` の中か確認する
2. 次のコマンドを打ってから、もう一度 ステップ2〜5 を試す

```
cd ~/cloudshell_open/nutrition-app
```

（この1行をコピー → 貼り付け → Enter）

---

## まとめ（2つのコマンド）

打つのはこの2つだけです。1つずつ、コピー → 貼り付け → Enter です。

**1つ目：**
```
chmod +x CloudShellでGitHubデプロイ設定.sh
```

**2つ目：**
```
./CloudShellでGitHubデプロイ設定.sh
```
